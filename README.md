# 基于 ESP32 的转向遥控系统

基于 ESP32 的转向遥控系统，通过遥控器实现远程控制，配合 PID 闭环算法实现精确转向。

## 系统架构

```
遥控器(970-1970μs PWM) → ESP32(CH1引脚) → PID控制器 ← 电位器(ADC反馈)
                                                    ↓
                                              双路PWM输出( GPIO25/GPIO26 )
                                                    ↓
                                              电机驱动模块 → 转向执行机构
```

## 硬件配置

| 模块    | 型号/参数                      | 引脚/地址                       |
| ----- | -------------------------- | --------------------------- |
| 主控    | ESP32                      | -                           |
| 遥控接收机 | HOTRC F-06A / DS600        | GPIO16                      |
| ADC转换 | PCF8591                    | I2C地址 0x48 (SDA:21, SCL:22) |
| 电位器   | 防爆电位器 3309492103PJMA24-300 | -                           |
| 电机驱动  | MOS开关模块                    | GPIO25(右转), GPIO26(左转)      |

## PWM信号规格

- 频率: 50Hz
- 脉宽范围: 970μs \~ 1970μs
- 中值: 1470μs
- 死区: ±30μs

| PWM脉宽  | 映射值   | 动作   |
| ------ | ----- | ---- |
| 970μs  | -100% | 左转最大 |
| 1470μs | 0%    | 停止   |
| 1970μs | +100% | 右转最大 |

## 模块说明

### Controller

主控制器，实现 PID 位置闭环控制。

**PID参数:**

- KP = 2.0
- KI = 0.1
- KD = 0.5
- 输出限幅: ±100%

**特性:**

- 条件积分: 仅当误差<20且未饱和时累积
- 积分分离: 大误差时停止积分，防止超调
- 积分限幅: ±30

### Controller\_test

与 Controller 相同，但启用双路 PWM 电机输出。

### ADC\_Collect

PCF8591 ADC 模块读取程序，用于标定电位器零点。

### PWM\_read

解析 HOTRC F-06A 遥控接收机的 PWM 信号。

### motor\_100

单通道 PWM 电机控制，支持串口输入 0\~100 控制占空比。

### motor\_100\_test

双通道 PWM 电机控制，支持串口同时控制两路电机 (格式: `50 80`)。

### uart

串口回传测试程序。

## 开发演进

模块按功能从基础到复杂演进:

```
PWM_read (读取遥控) → motor_100_test (验证PWM输出)
                              ↓
ADC_Collect (标定电位器) → Controller (PID算法验证) → Controller_test (最终成品)
```

**最终版本判定依据 Controller\_test:**

| 成熟度  | 文件                              | 说明                |
| ---- | ------------------------------- | ----------------- |
| ⭐ 基础 | PWM\_read / ADC\_Collect / uart | 单功能测试             |
| ⭐⭐   | motor\_100 / motor\_100\_test   | 电机驱动验证            |
| ⭐⭐⭐  | Controller                      | PID算法验证(仅串口输出)    |
| ⭐⭐⭐⭐ | **Controller\_test**            | **完整闭环 + 实际电机输出** |

Controller\_test 功能最完整，包含:

- 遥控 PWM 信号解析
- ADC 位置反馈
- PID 闭环控制算法
- 双路 PWM 电机实际输出

## 使用方法

1. **硬件连接**
   - 遥控接收机 CH1 输出连接 ESP32 GPIO16
   - PCF8591 模块连接 ESP32 I2C (SDA:21, SCL:22)
   - 电位器连接 PCF8591 AD0
   - 电机驱动模块连接 GPIO25, GPIO26
2. **控制器校准**
   - 使用 ADC\_Collect 读取电位器两端 ADC 值
   - 修改 Controller/Controller\_test 中的 ADC\_LEFT, ADC\_RIGHT 参数
3. **烧录程序**
   - 使用 Arduino IDE 或 PlatformIO 编译烧录
   - 波特率: 115200
4. **调试监控**
   - 打开串口监视器查看实时数据
   - 格式: `目标:% | 实际:% | 误差 | 积分项 | PID输出 | 动作 | ADC`

## 文件结构

```
├── Controller/           # 主控制器 (PID闭环)
├── Controller_test/      # 控制器测试 (带电机输出)
├── ADC_Collect/          # ADC采集测试
├── PWM_read/             # PWM信号读取
├── motor_100/            # 单路电机控制
├── motor_100_test/       # 双路电机控制
├── uart/                 # 串口回传测试
└── mos开关、遥控器、ad模块、电位器使用说明/   # 组件说明书
```

## 通信协议

- 串口波特率: 115200
- 数据格式: ASCII文本，每行一个数据点
- 更新频率: 20Hz (50ms间隔)
